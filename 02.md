## Theory  
An LED (Light Emitting Diode) glows when a certain amount of current flows through it in forward direction. We control LEDs using Arduino by sending HIGH or LOW signals through digital pins.  
To vary brightness, Arduino uses Pulse Width Modulation (PWM), where a digital pin switches between HIGH and LOW quickly. The average ON time (duty cycle) controls brightness. PWM is useful for applications like light dimming, motor speed control, and audio generation.  

Key functions used:  
- `pinMode(pin, OUTPUT)` ‚Üí defines output pin  
- `digitalWrite(pin, HIGH/LOW)` ‚Üí LED ON/OFF  
- `analogWrite(pin, value)` ‚Üí PWM brightness control (0‚Äì255)  
- `delay(ms)` ‚Üí creates time interval

## Circuit Explanation

- LED negative (short leg) is connected to GND through a 220Œ© resistor.
- For PWM fading, we use PWM-capable pins (~ symbol) like D9.
- Arduino controls LED blinking and brightness using digitalWrite() and analogWrite() functions.

---

## Arduino Code 1: Blinking LED
```cpp
/*
   Blink LED
   Turns LED ON for one second and OFF for one second repeatedly.
*/
void setup() {
  pinMode(8, OUTPUT);  // Set pin 8 as OUTPUT
}
void loop() {
  digitalWrite(8, HIGH);  // Turn ON
  delay(1000);           // 1 second delay
  digitalWrite(8, LOW);   // Turn OFF
  delay(1000);
}
````

---

## Arduino Code 2: LED Fading using PWM

```cpp
int ledPin = 9; // PWM pin
void setup() {
  pinMode(ledPin, OUTPUT);
}
void loop() {
  for(int fade = 0; fade <= 255; fade += 5){
    analogWrite(ledPin, fade); // increase brightness gradually
    delay(30);
  }
  for(int fade = 255; fade >= 0; fade -= 5){
    analogWrite(ledPin, fade); // decrease brightness gradually
    delay(30);
  }
}
```

## Result

The LED successfully blinks at a fixed rate and its brightness smoothly fades IN and OUT using PWM. This confirms the concept of digital output and analog-like PWM.
