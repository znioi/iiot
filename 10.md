## Aim

## Theory  
16x2 LCD (Hitachi HD44780 controller) displays 16 characters per row and 2 lines. It uses a **parallel communication interface** controlled by RS, RW, EN and data pins D0‚ÄìD7.  

üìå Important Pins  
| Pin | Function |
|-----|----------|
| RS | Command/Data select (0 ‚Üí Command, 1 ‚Üí Data) |
| RW | Read/Write (0 ‚Üí Write, 1 ‚Üí Read) |
| EN | Enable (Triggers data latch) |
| D4‚ÄìD7 | Data bits used in 4-bit mode for characters |
| VSS, VDD | Power pins |
| Vo | Contrast control using 10K pot |
| A, K | Backlight LED pins |

Arduino uses **LiquidCrystal library** to send formatted text easily:  
- `lcd.begin(16,2)` ‚Üí Initialize LCD  
- `lcd.setCursor(col,row)` ‚Üí Position cursor  
- `lcd.print("text")` ‚Üí Display characters  

Modes of LCD communication:  
| Mode | Pins Required | Usage |
|------|--------------|------|
| 8-bit | 11 pins | Faster but more wiring |
| 4-bit | 7 pins | Most commonly used |

## Circuit Description  
üìå LCD Interface pins:  
- RS ‚Üí **Pin 12**  
- EN ‚Üí **Pin 11**  
- D4 ‚Üí **Pin 10**  
- D5 ‚Üí **Pin 9**  
- D6 ‚Üí **Pin 8**  
- D7 ‚Üí **Pin 7**  
- R/W ‚Üí GND  
- VSS ‚Üí GND  
- VDD ‚Üí +5V  
- Vo ‚Üí Middle terminal of 10K pot  
- Pot ends ‚Üí +5V and GND  
- Backlight A ‚Üí +5V  
- Backlight K ‚Üí GND  

üìå LDR interface  
- LDR + 10K resistor ‚Üí Voltage divider  
- Middle junction ‚Üí **A0** of Arduino

## Code 2 ‚Äî LCD Display with LDR (Light Sensor)

```cpp
#include <LiquidCrystal.h>

LiquidCrystal lcd(12, 11, 10, 9, 8, 7);

void setup() {
  lcd.begin(16, 2);
  lcd.setCursor(0, 0);
  lcd.print("Light Status:");
}

void loop() {
  int sensorValue = analogRead(A0);

  lcd.setCursor(0, 1); // second line
  if (sensorValue > 700) {
    lcd.print("Bright    ");
  } else {
    lcd.print("Dark      ");
  }

  delay(500);
}
```

* LCD displays **Bright/Dark** depending on ambient light.
