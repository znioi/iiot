
# Experiment No. 6  
## Title  
Temperature & Humidity Measurement using DHT Sensor (DHT11)

---

## Aim  
To interface a DHT11 sensor with Arduino UNO and measure real-time temperature and humidity values using digital output.

---

## Theory  
DHT11 is a low-cost digital environmental sensor capable of measuring **temperature** and **humidity** simultaneously. It has an internal **capacitive humidity sensor** and **NTC thermistor** for temperature.  
A built-in **8-bit microcontroller** processes sensor readings and sends calibrated digital output to Arduino.  

Key Features:  
| Parameter | DHT11 |
|----------|-------|
| Humidity Range | 20% ‚Äì 90% RH |
| Temperature Range | 0¬∞C ‚Äì 50¬∞C |
| Output Type | Digital signal |
| Accuracy | ¬±5% RH, ¬±2¬∞C |

DHT11 Pin Description:  
| Pin | Function |
|-----|----------|
| 1 | VCC (3.3V‚Äì5V) |
| 2 | DATA ‚Üí Arduino pin 2 |
| 3 | NC (Not Connected) |
| 4 | GND |

DHT uses a **single-wire communication** protocol, so only one digital pin required.

## Arduino Code for DHT11 (Final Correct Working Code)
```cpp
#include <DHT.h>

#define DHTPIN 2       // Sensor connected to pin 2
#define DHTTYPE DHT11  // Sensor Type DHT11

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  Serial.println("DHT11 Sensor Test");
  dht.begin();
}

void loop() {
  delay(2000); // Sensor reading delay

  float humidity = dht.readHumidity();
  float temperature = dht.readTemperature();

  if (isnan(humidity) || isnan(temperature)) {
    Serial.println("Error Reading DHT11 Data!");
    return;
  }

  Serial.print("Humidity: ");
  Serial.print(humidity);
  Serial.print("%  Temperature: ");
  Serial.print(temperature);
  Serial.println("¬∞C");
}
````

---

## Output

The Serial Monitor displays real-time temperature in ¬∞C and humidity in %RH every 2 seconds.

## Result

The DHT11 sensor readings successfully display humidity and temperature values on the Serial Monitor, proving successful interfacing and measurement.
