## Theory  
Functions are reusable blocks of code that perform specific tasks. They help in modular programming, reduce code repetition, and improve readability.  

There are two types of functions in Arduino:
1. **Built-in Functions** ‚Äì Provided by Arduino (e.g., pinMode(), digitalWrite(), delay(), Serial.print())  
2. **User-Defined Functions** ‚Äì Created by the programmer to perform repeated tasks  

Standard Function Structure:
```cpp
return_type functionName(parameters) {
   // Code statements
   return value;  // optional
}
````

Every Arduino sketch must contain:

* `setup()` ‚Üí Runs only once
* `loop()` ‚Üí Runs continuously

## Code Example 1: Using Built-in Functions

```cpp
void setup() {
  Serial.begin(9600);
  pinMode(8, OUTPUT);
}

void loop() {
  digitalWrite(8, HIGH);
  Serial.println("LED ON");
  delay(1000);

  digitalWrite(8, LOW);
  Serial.println("LED OFF");
  delay(1000);
}
```

---

## Code Example 2: Using a User-Defined Function

```cpp
int led = 9;

void blinkLED(int timeDelay) {
  digitalWrite(led, HIGH);
  delay(timeDelay);
  digitalWrite(led, LOW);
  delay(timeDelay);
}

void setup() {
  pinMode(led, OUTPUT);
}

void loop() {
  blinkLED(500); // Calling user-defined function
}
```

---

## Result

The LED blinks at a constant interval repeatedly by calling the custom function. This demonstrates the use of both built-in and user-defined functions in Arduino programming.
