## Theory  
LDR (Light Dependent Resistor) is a passive electronic component whose resistance decreases when light intensity increases. When placed in a voltage divider network with a fixed resistor, a variable analog voltage is produced at their junction.  
Arduino contains a **10-bit ADC** which converts analog input (0‚Äì5V) into **digital values 0‚Äì1023**. Based on the amount of light falling on LDR, Arduino reads different values and controls output devices like LEDs.

Example behavior:  
| Light Intensity | LDR Resistance | Analog Reading |
|----------------|----------------|----------------|
| Bright Light | Low resistance | High value (800‚Äì1023) |
| Dark | High resistance | Low value (0‚Äì400) |

LDR connections:
- **AO pin** ‚Üí Analog input A0 of Arduino  
- **Voltage divider** formed using 10k resistor + LDR  
- LED used as output controlled by analog threshold  

---

## Circuit Explanation  
- Connect LDR and 10k resistor in voltage divider between 5V and GND  
- Junction goes to **A0**  
- LED_BUILTIN (pin 13) is used as indicator  
- Bright light ‚Üí LED OFF  
- Dark ‚Üí LED ON  

---

## Arduino Code
```cpp
void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int sensorValue = analogRead(A0);
  Serial.println(sensorValue); // for testing in Serial Monitor

  if (sensorValue > 700) {
    digitalWrite(LED_BUILTIN, HIGH);
  } else {
    digitalWrite(LED_BUILTIN, LOW);
  }
  delay(10);
}
````

---

## Result

The LED glows when the LDR detects darkness or low light conditions and turns OFF in bright light. The Serial Monitor displays analog readings showing real-time light measurements.
